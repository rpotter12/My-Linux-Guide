# nproc
nproc command shows the number of processing units present to the current process. Itâ€™s output may be less than the number of online processors on a system.

## Syntax
`nproc [OPTION]...`

## Description
```
       --all  print the number of installed processors

       --ignore=N
              if possible, exclude N processing units

       --help display this help and exit

       --version
              output version information and exit
```
